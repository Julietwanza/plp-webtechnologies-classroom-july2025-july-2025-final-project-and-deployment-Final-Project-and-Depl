## 🚀 Deployment

You can deploy this website for free on platforms like:

- **GitHub Pages**  
- **Netlify**

### Example (GitHub Pages):
1. Push your project to a GitHub repository.  
2. Go to **Settings > Pages**.  
3. Under *Branch*, select `main` and `/root`.  
4. Save – your site will be live at `https://your-username.github.io/sdg7-website/`.
